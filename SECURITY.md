# Security and Responsible Use

This project processes electoral data and generates review-prioritization signals.

Do not use the system output as a standalone accusation, legal conclusion or public claim of confirmed fraud.

The system is intended for:

- exploratory analysis,
- evidence organization,
- review prioritization,
- human analyst support.

The system is not intended for:

- final legal determination,
- public accusation of individuals or organizations,
- automated decision-making without human review.

If deployed beyond local experimentation, add authentication, authorization, audit logs, backup strategy and secure secret management.
