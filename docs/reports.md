# Reports

Reports are designed for reviewers.

## Case report

Contains:

- readable case information,
- priority and review score,
- executive summary,
- technical summary,
- structured evidence,
- score components,
- limitations,
- recommended next steps,
- traceability.

Technical IDs are hidden in traceability sections.

## Executive report

Contains a summary of the analysis execution and review cases generated.

## Report files

HTML reports are exported to:

```text
data/exports/
```

They can also be opened through:

```text
GET /reports/{report_id}/html
```
