# Methodology and Limits

The application identifies anomalous patterns and prioritizes cases for review.

It can say:

- a table has unusual turnout,
- an electoral option has unusually high concentration,
- a table differs from its territorial context,
- a case has high review priority,
- human or documentary review is recommended.

It must not say:

- fraud is confirmed,
- a person or organization committed fraud,
- a result is false,
- manipulation is proven,
- the evidence is legally conclusive.

Statistical signals can justify review, but they do not prove intent, authorship or criminal conduct.

Claude, when enabled, only drafts narrative explanations from structured context. It does not compute metrics, create alerts, modify scores or decide whether fraud occurred.
